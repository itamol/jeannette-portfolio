# Portfolio Profesional - Jeannette Guardia

## Descripción

Portfolio web profesional de **Jeannette Guardia**, Analista Funcional con experiencia en gestión de proyectos, análisis de requerimientos, UX/UI y coordinación entre equipos técnicos y comerciales.

## Características

✨ **Diseño Moderno y Responsivo**
- Interfaz limpia y profesional
- Totalmente responsiva para dispositivos móviles, tablets y desktop
- Animaciones suaves y transiciones elegantes

🎯 **Secciones Principales**
- **Inicio (Hero):** Presentación impactante con call-to-action
- **Perfil:** Información personal y de contacto
- **Habilidades:** Tarjetas interactivas con competencias técnicas y soft skills
- **Experiencia:** Timeline visual de trayectoria profesional
- **Formación:** Certificaciones y cursos completados
- **Contacto:** Enlaces directos para comunicación

📱 **Funcionalidades**
- Navegación sticky con menú hamburguesa para móviles
- Scroll suave entre secciones
- Animaciones de aparición al hacer scroll
- Efectos hover en tarjetas
- Integración con LinkedIn y email

## Cómo Usar

### Localmente

1. Descarga los archivos del portfolio
2. Abre `index.html` en tu navegador web
3. ¡Disfruta del portfolio!

### Servidor Local (Desarrollo)

```bash
cd jeannette-portfolio
python3 -m http.server 8000
```

Luego accede a `http://localhost:8000` en tu navegador.

### Desplegar en Línea

Puedes desplegar este sitio en plataformas como:
- **GitHub Pages** (gratuito)
- **Netlify** (gratuito con opciones premium)
- **Vercel** (gratuito con opciones premium)
- **Firebase Hosting** (gratuito)

## Personalización

### Cambiar Colores

Edita las variables CSS en `styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #f59e0b;
    /* ... más variables */
}
```

### Actualizar Contenido

Modifica el contenido directamente en `index.html`:
- Información de contacto
- Experiencia laboral
- Habilidades
- Formación académica

### Agregar Nuevas Secciones

1. Añade el HTML en `index.html`
2. Crea los estilos correspondientes en `styles.css`
3. Agrega interactividad en `script.js` si es necesario
